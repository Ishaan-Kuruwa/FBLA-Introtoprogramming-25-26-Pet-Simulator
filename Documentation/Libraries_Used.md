**The Libraries used in this project include:**
*NO TEMPLATES WERE USED IN THIS PROJECT

**Random**
-   Purpose: Used to make game more interactive through "random" increases in attributes and games
-   Setup: Random library comes with Python installation
-   Documentation: [https://docs.python.org/3/library/random.html](url)


**Matplotlib**   
-   Purpose: Used to display images of pet
-   Setup: "pip install matplotlib"
-   Documentation: [https://matplotlib.org/stable/index.html](url)

**Os**
-   Purpose: Clear screen function
-   Setup: Os library comes with Python installation
-   Documentation: [https://docs.python.org/3/library/os.html](url)
  
**Time**
-   Purpose: Clear screen function, make UI interactive
-   Setup: Time library comes with Python installation
-   Documentation: [https://docs.python.org/3/library/time.html](url)

**Pillow**
-   Purpose: Display images, works with matplotlib
-   Setup: "pip install Pillow"
-   Documentation: [https://pillow.readthedocs.io/en/stable/](url)
