# FBLA-Introtoprogramming-25-26-Pet-Simulator - _VirtuPet_

Welcome to my Pet Simulator for Introduction to Programming in the FBLA High School Division 2025/2026 season!

This project is a python-based game to educate future pet-owners on how to care for a real life pet.

**My Vision**:
- Target
   Students
   Aspiring pet-owners
  
- Why
   Learn to take care of a pet through a fun and interactive python game
  
- What
   Python video game 
   Feed, care-for, buy items, and check financials for pet

**My Mission**:

- Responsibility
   Consistent Care
   Avoid neglect
   Respond to pet emotions
  
- Financial Literacy
   Food cost
   Vet visits
   Total expenses
  
- Decision-Making
   +/- to each move
   Choose wisely

**Configuration Details:**
- First, Please download all libraries in "Libraries_Used.md" using "pip install" or other commands.
- Then, please download the "Game" file or copy and paste the code into your code environment.
- OPTIONAL: Download the Images folder labeled "IntrotoProgramming25-26Images".
- Finally, You are ready to play!
